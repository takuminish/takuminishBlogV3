## 基本的な型

TypeScriptには以下の基本型があります。

- `string` - 文字列
- `number` - 数値
- `boolean` - 真偽値

```typescript
const name: string = "TypeScript";
const version: number = 5;
const isAwesome: boolean = true;
```
